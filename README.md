# Testi letterari con errori da correggere

Questo repository contiene parti di testi letterari
famosi che però contengono degli errori. Può essere usato
per esercitarsi con alcuni comandi _git_, ad esempio
alcuni studenti risolvono i problemi di un testo
mentre altri risolvono i problemi di un altro testo
e il lavoro viene poi unito (_merge_).

Si può lavorare in parallelo sia su file differenti
sia sugli stessi file, a patto di non creare conflitti,
cioè non fare 2 modifiche allo stesso file contrastanti.

## Testi disponibili

Sono disponibili i seguenti testi, ognuno con
un elenco di errori _ben evidenziati_ nel testo
e le indicazioni di come correggerli:

- Dante, [La Divina Commedia](divina_commedia/ERRORI.md)
- Manzoni, [I Promessi Sposi](promessi_sposi/ERRORI.md)
- Verne, [Ventimila Leghe sotto ai Mari](ventimila_leghe/ERRORI.md)
- Verne, [Il Giro del Mondo in 80 Giorni](giro_mondo/ERRORI.md)
